# JS Review

## Description

This project is a review of Javascript concepts we have already covered in class to ensure everyone is on the same page as move forward to more advanced concepts.

## Instructions

### Setup

1. Fork this repo
2. Clone your forked repo

### Problems

1. For each directory - open the `index.html` file in your browser and open the console. You will see a series of failing tests. Your job is to make the tests pass by writing the correct code in the directories `practice.js` file. You can check your progress by refreshing the browser window.
